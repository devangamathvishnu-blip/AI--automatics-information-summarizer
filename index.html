AI News Summarizer

Automated AI-powered News Summarization using n8n & Google Gemini

Project Overview

This project automatically collects news from RSS feeds and Search APIs, summarizes them using Google Gemini AI, and sends daily summaries to the user's Gmail.

Features

  • Automatic News Collection
  • RSS Feed Integration
  • Search API Integration
  • Google Gemini AI Summarization
  • Automated Gmail Delivery
  • Daily Scheduled Workflow

Technology Stack

n8n • Google Gemini AI • RSS Feed • Search API • Gmail
